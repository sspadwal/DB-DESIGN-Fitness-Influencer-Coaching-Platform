Fitness Coaching Platform - ER Diagram

This project is a database design for an online fitness coaching platform where trainers manage multiple clients and provide structured coaching.

The system supports client onboarding, fitness plans, plan enrollment, payments, session scheduling, and progress tracking.

A separate enrollment (join) entity is used to track which user joined which plan along with start and end dates.

Weekly check-ins and trainer feedback are modeled separately to track user progress over time.

This design follows real-world requirements like multiple plan purchases, trainer-client relationships, and scalable tracking of sessions and progress.
